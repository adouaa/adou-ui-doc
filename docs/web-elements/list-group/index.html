List Group 列表

ListGroup 组件用于展示一个可分列的列表，支持单击和双击事件、默认选中、动态渲染等功能。

1. 基本使用​

效果示例：

点击查看代码
import React from "react";
import List from "@site/src/components/web-elements/adou-list";
import { data } from "./data";

interface BasicProps {}

const Basic = ({}: BasicProps) => {
  return (
    <div className="basic-wrapper p-2">
      <List isTree data={data}></List>
    </div>
  );
};

export default Basic;

2. 分列​

  • 通过设置 lineBreak 为 true 组件可以分列显示。
  • 分列有三种方法，分别可以通过 listPerColumn、columnMaxHeight 属性以及 控制 父容器的高度 来实现。
  • listPerColumn 属性优先级最高，当设置 listPerColumn 时，其他属性失效。
  • columnMaxHeight 属性和 父容器的高度 优先级次之，通过设置 itemHeight 属性配合实现。

效果示例：

方式一: 通过 listPerColumn 属性
方式二: 通过 columnMaxHeight 属性
方式三: 通过设置 父容器的高度，itemHeight = 60
点击查看代码
import React from "react";
import ListGroup from "@site/src/components/web-elements/adou-list-group";
import { data } from "./data";

interface SplitProps {}

const Split = ({}: SplitProps) => {
  return (
    <div className="split-wrapper">
      <h5 className="mb-2">方式一: 通过 listPerColumn 属性</h5>
      <ListGroup lineBreak listPerColumn={3} data={data}></ListGroup>
      <h5 className="my-2">方式二: 通过 columnMaxHeight 属性</h5>
      <ListGroup lineBreak columnMaxHeight={260} data={data}></ListGroup>
      <h5 className="my-2">方式三: 通过设置 父容器的高度，itemHeight = 60</h5>
      <div style={{ height: "260px" }}>
        <ListGroup lineBreak itemHeight={60} data={data}></ListGroup>
      </div>
    </div>
  );
};

export default Split;

3. 取消点击激活​

组件默认支持点击激活，通过设置 activeOnClick 为 false 可以取消点击激活。

效果示例：

点击查看代码
import React from "react";
import ListGroup from "@site/src/components/web-elements/adou-list-group";
import { data } from "./data";

interface ActiveOnClickProps {}

const ActiveOnClick = ({}: ActiveOnClickProps) => {
  return (
    <div className="active-on-click-wrapper">
      <ListGroup activeOnClick={false} data={data}></ListGroup>
    </div>
  );
};

export default ActiveOnClick;

4. 最大高度​

设置 maxHeight 属性，可以给组件设置一个最大高度，超出滚动

效果示例：

点击查看代码
import React from "react";
import ListGroup from "@site/src/components/web-elements/adou-list-group";
import { data } from "./data";

interface MaxHeightProps {}

const MaxHeight = ({}: MaxHeightProps) => {
  return (
    <div className="basic-wrapper">
      <ListGroup maxHeight={"300px"} data={data}></ListGroup>
    </div>
  );
};

export default MaxHeight;

5. 选中第一条​

设置 defaultFirst 属性，可以给组件设置一个最大高度，超出滚动

效果示例：

点击查看代码
import React from "react";
import ListGroup from "@site/src/components/web-elements/adou-list-group";
import { data } from "./data";

interface DefaultFirstProps {}

const DefaultFirst = ({}: DefaultFirstProps) => {
  return (
    <div className="default-first-wrapper">
      <ListGroup defaultFirst data={data}></ListGroup>
    </div>
  );
};

export default DefaultFirst;

6. 多选​

设置 multiple 可以开启多选，并且可以通过 activeList 设置默认选中项。

效果示例：

点击查看代码
import React, { useState } from "react";
import ListGroup from "@site/src/components/web-elements/adou-list-group";
import { data } from "./data";

interface MultipleProps {}

const Multiple = ({}: MultipleProps) => {
  const [selectedList, setSelectedList] = useState<any[]>([
    {
      value: "2",
    },
    { value: "4" },
    { value: "6" },
  ]);
  return (
    <div className="multiple-wrapper">
      <ListGroup multiple activeList={selectedList} data={data}></ListGroup>
    </div>
  );
};

export default Multiple;

7. 自定义渲染​

通过设置共用的 render 属性或者每一项的 render 属性可以自定义渲染每一项数据。

效果示例：

点击查看代码
import React from "react";
import ListGroup from "@site/src/components/web-elements/adou-list-group";

interface RenderProps {}

const Render = ({}: RenderProps) => {
  const data = [
    { label: "Item 1", value: "1" },
    {
      label: "Item 2",
      value: "2",
      render: () => (
        <a style={{ textDecoration: "underline", color: "red" }}>666</a>
      ),
    },
    { label: "Item 3", value: "3" },
    { label: "Item 4", value: "4" },
    { label: "Item 5", value: "5" },
    { label: "Item 6", value: "6" },
    { label: "Item 7", value: "7" },
    { label: "Item 8", value: "8" },
    { label: "Item 9", value: "9" },
    { label: "Item 10", value: "10" },
  ];

  return (
    <div className="render-wrapper">
      <ListGroup
        render={(item: any, labelKey: string) => (
          <span className="text-warning">{item[labelKey]}</span>
        )}
        data={data}
      ></ListGroup>
    </div>
  );
};

export default Render;

属性​

属性名类型默认值描述
multipleboolean无是否开启多选。
itemHeightany40每个列表项的高度，用于计算列数。
columnMaxHeightany无每列的最大高度。
lineBreakboolean无是否换行显示。
filesPerColumnnumber无每列展示的文件数量，优先级高于 columnMaxHeight 和 itemHeight。
columnsnumber2展示的列数。
heightany无列表的高度。
maxHeightany无列表的最大高度。
canActivebooleantrue是否允许激活状态。
externalClassNamestring无外部类名。
noWrapboolean无是否不换行。
defaultFirstbooleanfalse是否默认激活第一个列表项。
dataany[]无列表数据。
activeListany无当前激活的列表项。
labelKeystring"label"用于显示标签的键名。
valueKeystring"value"用于标识值的键名。
typestring"primary"激活状态的样式类型。
onItemClick(item?: any) => void无列表项点击事件的回调。
onItemDoubleClick(item?: any) => void无列表项双击事件的回调。
renderany无自定义渲染函数。
