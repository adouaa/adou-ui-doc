`HorizonScrollView` 是一个水平滚动视图组件，支持左右滚动按钮，适用于需要水平滚动的内容展示。。

## 基本使用

传入 `children` 即可

## Props

| 属性名   | 类型   | 默认值 | 描述           |
| -------- | ------ | ------ | -------------- |
| children | `any`  | 无     | 滚动视图的内容 |
