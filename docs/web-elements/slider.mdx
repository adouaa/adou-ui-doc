`Slider` 滑块组件用于在指定的数值区间内进行选择。支持单滑块和双滑块（范围选择）模式，可以设置步长、标记点，并支持输入框精确控制。

## 1. 基本使用

最基础的滑块用法，通过拖动滑块选择数值。

**效果示例**：

<Card>
<Basic />
<details className="mt-3">
<summary>点击查看代码</summary>
```jsx
import Slider from "@site/src/components/web-elements/adou-slider";
import React from "react";

const Basic = () => {
  return (
    <Slider
      min={0}
      max={100}
      onChange={(value, percent) => console.log(value, percent)}
    />
  );
};

export default Basic;
```

## 属性

| 属性名 | 类型 | 默认值 | 描述 |
|---|---|---|---|
| min | `number` | `0` | 最小值 |
| max | `number` | `100` | 最大值 |
| step | `number` | - | 步长，取值必须大于 0 |
| value | `number \| number[]` | - | 滑块的值，范围模式时为数组 |
| range | `boolean` | `false` | 是否启用范围选择模式 |
| showStops | `boolean` | `true` | 是否显示步长点 |
| showInput | `boolean` | `false` | 是否显示输入框 |
| sliderWidth | `string \| number` | - | 滑块宽度 |
| marks | `MarkItem[]` | - | 标记点配置 |
| onChange | `(value: number \| number[], percent: number) => void` | - | 值改变时的回调 |

## 特性

1. **多模式支持**：
   - 支持单滑块模式
   - 支持范围选择模式
   - 支持输入框精确控制

2. **标记功能**：
   - 支持自定义标记点
   - 支持步长显示
   - 支持标记文本自定义样式

3. **交互优化**：
   - 拖动时实时显示数值
   - 键盘支持
   - 点击滑动区域跳转

4. **样式定制**：
   - 可自定义滑块宽度
   - 支持标记点样式定制
   - 响应式设计

## 注意事项

1. **取值范围**：
   - min/max 需要合理设置
   - step 需要能被 (max-min) 整除

2. **范围模式**：
   - range 模式下不支持输入框
   - 两个滑块不能交叉

3. **性能考虑**：
   - marks 数量不宜过多
   - 避免频繁更新 value
