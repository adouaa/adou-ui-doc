`Cascader` 是一个级联选择组件，支持多级数据选择，适用于需要从多个层级中选择数据的场景。

## 1. 基本使用

**效果示例**：

<Cascader options={[{ label: 'Option 1', value: '1', children: [{ label: 'Option 1-1', value: '1-1' }] }]} />

## 2. `options`

通过 `options` 属性，可以设置级联选择的数据源。

### 2.1 `options`

**介绍**：
`options` 属性用于设定级联选择的数据源，数据源应为一个对象数组，每个对象可以包含 `label`、`value` 和 `children` 属性。

**效果示例**：

<Cascader options={[{ label: 'Option 1', value: '1', children: [{ label: 'Option 1-1', value: '1-1' }] }]} />

## 3. `defaultValue`

`defaultValue` 属性用于设置级联选择的默认选中值。

* 类型：`string[]`

**效果示例**：

<Cascader options={[{ label: 'Option 1', value: '1', children: [{ label: 'Option 1-1', value: '1-1' }] }]} defaultValue={['1', '1-1']} />

## 属性

| 属性名 | 类型 | 可选值 | 描述 |
| ---- | ---- | ---- | ---- |
| `width` | `any` | - | 设置组件的宽度。 |
| `inline` | `boolean` | - | 控制组件是否为行内元素。 |
| `activeColor` | `{ font: string; bgc: string }` | - | 设置选中项的字体和背景颜色。 |
| `returnType` | `string` | - | 设置返回值的类型。 |
| `required` | `boolean` | - | 控制组件是否为必填项。 |
| `name` | `string` | - | 设置组件的名称。 |
| `label` | `any` | - | 设置组件的标签。 |
| `errMsg` | `string` | - | 设置错误信息。 |
| `isFormItem` | `boolean` | - | 控制组件是否为表单项。 |
| `labelColor` | `string` | - | 设置标签的颜色。 |
| `showLabel` | `boolean` | - | 控制是否显示标签。 |
| `labelWidth` | `any` | - | 设置标签的宽度。 |
| `labelPosition` | `"center" | "left-top" | "top"` | - | 设置标签的位置。 |
| `clearable` | `boolean` | - | 控制是否可以清除选中项。 |
| `valueKey` | `string` | - | 设置值的键名。 |
| `labelKey` | `string` | - | 设置标签的键名。 |
| `filterable` | `boolean` | - | 控制是否可以过滤选项。 |
| `options` | `any` | - | 设置级联选择的数据源。 |
| `defaultValue` | `string[]` | - | 设置默认选中值。 |
| `onChange` | `(value: string[]) => void` | - | 选中值变化时的回调函数。 |
| `onFormDataChange` | `(key: string, value: any) => void` | - | 表单数据变化时的回调函数。 |
