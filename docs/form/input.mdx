## 2. 三种大小

通过 `size` 属性可以改变输入框的大小，可选值为 `sm`、`default` 和 `lg`，分别对应`32px`、`40px` 和 `48px`，默认值为 `default`。

**效果示例**：

<Card>
<Size />
<details className="mt-3">
<summary>点击查看代码</summary>
```jsx
import React from "react";
import Input from "@site/src/components/adou-new-form/adou-Input";
const SizeExample = () => {
  return (
    <div className="size-wrapper">
      <div className="sm-input">
        <Input size="sm" placeholder="请输入内容" />
      </div>
      <div className="default-input my-2">
        <Input placeholder="请输入内容" />
      </div>
      <div className="lg-input">
        <Input size="lg" placeholder="请输入内容" />
      </div>
    </div>
  );
};
export default SizeExample;
```

</details>
</Card>

## 3. 三种形式

通过 `varient` 属性可以改变输入框的展示形式，可选值有 `outlined`、`filled`、`borderless`，分别对应`正常边框`、`填充色`（默认没有边框，在有 `addonBefore` 或者 `addonAfter` 的时候有默认边框色）和 `无边框`，默认值为 `outlined`。

**效果示例**：

<Card>
<Varient />
<details className="mt-3">
<summary>点击查看代码</summary>
```jsx
import React from "react";
import Input from "@site/src/components/adou-new-form/adou-Input";
const SizeExample = () => {
  return (
    <div className="size-wrapper">
      <div className="sm-input">
        <Input size="sm" placeholder="请输入内容" />
      </div>
      <div className="default-input my-2">
        <Input placeholder="请输入内容" />
      </div>
      <div className="lg-input">
        <Input size="lg" placeholder="请输入内容" />
      </div>
    </div>
  );
};
export default SizeExample;
```
</details>
</Card>

## 5. 文本域

用于多行输入，通过 `rows` 属性可以设置文本域的行数，默认值为 `2`。

**效果示例**：

<Card>
<TextArea />
<details className="mt-3">
<summary>点击查看代码</summary>
```jsx
import React from "react";
import TextArea from "@site/src/components/adou-new-form/adou-textarea";

const TextAreaExample = () => {
return (

<div className="text-area-wrapper">
<TextArea defaultValue="这是一段文字"></TextArea>
</div>
);
};
export default TextAreaExample;
```
</details>
</Card>

## 6. 适应文本高度的文本域

`autoSize` 是一个对象，在指定最小行数和最大行数， `TextArea` 会自动适应文本的高度。

**效果示例**：

<Card>
<AutoSize />
<details className="mt-3">
<summary>点击查看代码</summary>
```jsx
import React, { useState } from "react";
import TextArea from "@site/src/components/adou-new-form/adou-textarea";

const AutoSizeExample = () => {
  return (
    <div className="auto-size-wrapper">
      <TextArea
        autoSize={{ minRows: 2, maxRows: 4 }}
        defaultValue="这是一段文字"
      ></TextArea>
    </div>
  );
};
export default AutoSizeExample;
```
</details>
</Card>

## 6. 前缀和后缀

通过 `prefix` 和 `suffix` 属性可以为输入框上添加 `前缀` 或 `后缀`。

**效果示例**：

<Card>
<PrefixAndSuffix />
<details className="mt-3">
<summary>点击查看代码</summary>
```jsx
import React, { useState } from "react";
import Input from "@site/src/components/adou-new-form/adou-Input";

const PrefixAndSuffixExample = () => {
  return (
    <div className="basic-wrapper">
      <div className="common">
        <Input prefix={"￥"} placeholder="前缀和后缀" />
      </div>
      <div className="icon mt-2">
        <Input
          prefix={<i className="fa fa-user"></i>}
          suffix={<i className="fa fa-search"></i>}
          placeholder="前缀和后缀图标"
        />
      </div>
    </div>
  );
};
export default PrefixAndSuffixExample;
```

</details>
</Card>

## Input Props

| 属性名                         | 类型                                                         | 默认值         | 描述                     |
| ------------------------------ | ------------------------------------------------------------ | -------------- | ------------------------ |
| inputStyle                     | `React.CSSProperties`                                        | 无             | 输入框的自定义样式       |
| prefix                         | `any`                                                        | 无             | 输入框前缀内容           |
| suffix                         | `any`                                                        | 无             | 输入框后缀内容           |
| addonBefore                    | `ReactNode \| string \| number`                              | 无             | 输入框前的附加内容       |
| addonAfter                     | `ReactNode \| string \| number`                              | 无             | 输入框后的附加内容       |
| varient                        | `'outlined' \| 'filled' \| 'borderless'`                     | `'outlined'`   | 输入框的样式变体         |
| valueKey                       | `string`                                                     | 无             | 值的键名                 |
| labelKey                       | `string`                                                     | 无             | 标签的键名               |
| wrapperWidth                   | `any`                                                        | 无             | 包裹层的宽度             |
| wrapperStyle                   | `React.CSSProperties`                                        | 无             | 包裹层的自定义样式       |
| commonSuffixContent            | `string`                                                     | 无             | 通用后缀内容             |

| defaultValue                   | `any`                                                        | 无             | 默认值                   |
| size                           | `'lg' \| 'default' \| 'sm'`                                  | 无             | 输入框的大小             |
| externalClassName              | `string`                                                     | 无             | 外部类名                 |
| prefixContent                  | `any`                                                        | 无             | 前缀内容                 |
