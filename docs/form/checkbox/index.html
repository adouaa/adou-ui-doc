Checkbox 复选框

Checkbox 是一个复选框组件，支持多选项选择，适用于需要从多个选项中选择一个或多个的场景。

1. 基本使用​

效果示例：

食物
查看代码
import React from "react";
import Checkbox from "adou-ui/Checkbox";

interface BasicProps {}

const Basic = ({}: BasicProps) => {
  return (
    <div className="basic-wrapper">
      <Checkbox
        label="食物"
        options={[
          { label: "Option 1", value: "1" },
          { label: "Option 2", value: "2" },
        ]}
      ></Checkbox>
    </div>
  );
};

export default Basic;


2. options​

通过 options 属性，可以设置复选框的选项。

2.1 options​

介绍： options 属性用于设定复选框的选项，数据源应为一个对象数组，每个对象包含 label 和 value 属性。

效果示例：

食物
查看代码
import { useState } from "react";
import React from "react";
import Checkbox from "adou-ui/Checkbox";

interface OptionsProps {}

const Options = ({}: OptionsProps) => {
  const [foods, setFoods] = useState<any[]>([
    { label: "苹果", value: "苹果" },
    { label: "香蕉", value: "香蕉" },
    { label: "橙子", value: "橙子" },
    { label: "西瓜", value: "西瓜" },
    { label: "葡萄", value: "葡萄" },
  ]);

  return (
    <div className="basic-wrapper">
      <Checkbox label="食物" options={foods}></Checkbox>
    </div>
  );
};

export default Options;



3. defaultValue​

defaultValue 属性用于设置复选框的默认选中值。

  • 类型：string | string[] | any[]

效果示例：

食物
查看代码
import React from "react";
import Checkbox from "adou-ui/Checkbox";

interface DefaultValueProps {}

const DefaultValue = ({}: DefaultValueProps) => {
  return (
    <div className="default-value-wrapper">
      <Checkbox
        label="食物"
        options={[
          { label: "Option 1", value: "1" },
          { label: "Option 2", value: "2" },
        ]}
      ></Checkbox>
    </div>
  );
};

export default DefaultValue;


4. inline​

inline 属性用于设置选项是横向排列还是纵向排列，默认为 true，横向排列。

效果示例：

食物
查看代码
import { useState } from "react";
import React from "react";
import Checkbox from "@site/src/components/adou-new-form/adou-checkbox";

interface InlineProps {}

const Inline = ({}: InlineProps) => {
  const [options, setOptions] = useState<any[]>([
    { label: "苹果", value: "苹果" },
    { label: "香蕉", value: "香蕉" },
    { label: "橙子", value: "橙子" },
    { label: "西瓜", value: "西瓜" },
    { label: "葡萄", value: "葡萄" },
  ]);

  return (
    <div className="inline-wrapper">
      <Checkbox
        label="食物"
        inline={false}
        options={options}
        defaultValue={["香蕉", "橙子"]}
      />
    </div>
  );
};

export default Inline;



5. suffixContent​

suffixContent 属性用于设置设置后缀内容。

效果示例：

食物
查看代码
import { useState } from "react";
import React from "react";
import Checkbox from "@site/src/components/adou-new-form/adou-checkbox";
import Button from "@site/src/components/adou-button";

interface SuffixContentProps {}

const SuffixContent = ({}: SuffixContentProps) => {
  const [options, setOptions] = useState<any[]>([
    { label: "苹果", value: "苹果" },
    { label: "香蕉", value: "香蕉" },
    { label: "橙子", value: "橙子" },
    { label: "西瓜", value: "西瓜" },
    { label: "葡萄", value: "葡萄" },
  ]);

  return (
    <div className="suffix-content-wrapper">
      <Checkbox
        label="食物"
        suffixContent={
          <Button type="warning" size="sm">
            选择食物
          </Button>
        }
        inline={false}
        options={options}
      />
    </div>
  );
};

export default SuffixContent;

6. 校验​

6.1 required​

required 属性用于设置是否需要校验表单。

6.2 errMsg​

errMsg 属性用于设置表单为空时提示的内容，默认是字段 label 或 name 的值 + 不能为空。

效果示例：

食物
查看代码
import { useState } from "react";
import React from "react";
import Checkbox from "@site/src/components/adou-new-form/adou-checkbox";

interface RequiredProps {}

const Required = ({}: RequiredProps) => {
  const [foods, setFoods] = useState<any[]>([
    { label: "苹果", value: "苹果" },
    { label: "香蕉", value: "香蕉" },
    { label: "橙子", value: "橙子" },
    { label: "西瓜", value: "西瓜" },
    { label: "葡萄", value: "葡萄" },
  ]);

  return (
    <div className="basic-wrapper">
      <Checkbox
        required
        errMsg="请选择一种食物哦~"
        label="食物"
        options={foods}
      ></Checkbox>
    </div>
  );
};

export default Required;

属性​

属性名类型可选值描述
widthany-设置组件的宽度。
valueKeystring-设置值的键名。
labelKeystring-设置标签的键名。
returnTypestringstr / obj-
suffixContentTypestring-设置后缀内容的类型。
suffixContentany-设置后缀内容。
namestring-设置组件的名称。
isFormItemboolean-控制组件是否为表单项，不是则没有下边距。
errMsgstring-设置错误信息。
labelWidthany-设置标签的宽度。
commonSuffixIconstring-设置通用后缀图标。
readOnlyboolean-控制组件是否为只读。
labelstring-设置组件的标签。
labelPositionstringcenter / left-top / top设置标签的位置，left-top 为左上
labelColorstring-设置标签的颜色。
requiredboolean-控制组件是否为必填项。
defaultValuestringstring[]any[]
externalClassNamestring-设置外部类名。
optionsany[]-设置复选框的选项。
inlineboolean-控制选项是否为行内显示。
wrapboolean-控制选项是否换行。
onChange(item: any) => void-选中值变化时的回调函数。
onFormDataChange() => void-表单数据变化时的回调函数。
onFieldChange() => void-字段变化时的回调函数。
onValidateField() => void-字段验证时的回调函数。
