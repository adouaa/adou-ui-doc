`Checkbox` 是一个多选框组件，支持多选项选择，适用于需要从多个选项中选择一个或多个的场景。

## 1. 基本使用

**效果示例**：
<Card>
    <Basic />

    <details className="mt-3">
    <summary>查看代码</summary>

```javascript
import React from "react";
import Checkbox from "adou-ui/Checkbox";

interface BasicProps {}

const Basic = ({}: BasicProps) => {
  return (
    <div className="basic-wrapper">
      <Checkbox
        label="食物"
        options={[
          { label: "Option 1", value: "1" },
          { label: "Option 2", value: "2" },
        ]}
      ></Checkbox>
    </div>
  );
};

export default Basic;


```

    </details>
</Card>


## 2. `options`

通过 `options` 属性，可以设置多选框的选项。

### 2.1 `options`

**介绍**：
`options` 属性用于设定多选框的选项，数据源应为一个对象数组，每个对象包含 `label` 和 `value` 属性。

**效果示例**：

<Card>
    <Options />

    <details className="mt-3">
    <summary>查看代码</summary>

```javascript
import { useState } from "react";
import React from "react";
import Checkbox from "adou-ui/Checkbox";

interface OptionsProps {}

const Options = ({}: OptionsProps) => {
  const [foods, setFoods] = useState<any[]>([
    { label: "苹果", value: "苹果" },
    { label: "香蕉", value: "香蕉" },
    { label: "橙子", value: "橙子" },
    { label: "西瓜", value: "西瓜" },
    { label: "葡萄", value: "葡萄" },
  ]);

  return (
    <div className="basic-wrapper">
      <Checkbox label="食物" options={foods}></Checkbox>
    </div>
  );
};

export default Options;



```

    </details>
</Card>

## 4. `inline`

`inline` 属性用于设置选项是横向排列还是纵向排列，默认为 `true`，横向排列。

**效果示例**：

<Card>
<Inline />

<details className="mt-3">
<summary>查看代码</summary>

```javascript
import { useState } from "react";
import React from "react";
import Checkbox from "@site/src/components/adou-new-form/adou-checkbox";

interface InlineProps {}

const Inline = ({}: InlineProps) => {
  const [options, setOptions] = useState<any[]>([
    { label: "苹果", value: "苹果" },
    { label: "香蕉", value: "香蕉" },
    { label: "橙子", value: "橙子" },
    { label: "西瓜", value: "西瓜" },
    { label: "葡萄", value: "葡萄" },
  ]);

  return (
    <div className="inline-wrapper">
      <Checkbox
        label="食物"
        inline={false}
        options={options}
        defaultValue={["香蕉", "橙子"]}
      />
    </div>
  );
};

export default Inline;



```

</details>
</Card>

## 5. `suffixContent`

`suffixContent` 属性用于设置设置后缀内容。

**效果示例**：

<Card>
<SuffixContent />

<details className="mt-3">
<summary>查看代码</summary>

```javascript
import { useState } from "react";
import React from "react";
import Checkbox from "@site/src/components/adou-new-form/adou-checkbox";
import Button from "@site/src/components/adou-button";

interface SuffixContentProps {}

const SuffixContent = ({}: SuffixContentProps) => {
  const [options, setOptions] = useState<any[]>([
    { label: "苹果", value: "苹果" },
    { label: "香蕉", value: "香蕉" },
    { label: "橙子", value: "橙子" },
    { label: "西瓜", value: "西瓜" },
    { label: "葡萄", value: "葡萄" },
  ]);

  return (
    <div className="suffix-content-wrapper">
      <Checkbox
        label="食物"
        suffixContent={
          <Button type="warning" size="sm">
            选择食物
          </Button>
        }
        inline={false}
        options={options}
      />
    </div>
  );
};

export default SuffixContent;
```

</details>

## 6. `校验`

### 6.1 `required`

`required` 属性用于设置是否需要校验表单。

### 6.2 `errMsg`

`errMsg` 属性用于设置表单为空时提示的内容，默认是字段 `label` 或 `name` 的值 + 不能为空。

**效果示例**：

<Card>
<Required />

<details className="mt-3">
<summary>查看代码</summary>

```javascript
import { useState } from "react";
import React from "react";
import Checkbox from "@site/src/components/adou-new-form/adou-checkbox";

interface RequiredProps {}

const Required = ({}: RequiredProps) => {
  const [foods, setFoods] = useState<any[]>([
    { label: "苹果", value: "苹果" },
    { label: "香蕉", value: "香蕉" },
    { label: "橙子", value: "橙子" },
    { label: "西瓜", value: "西瓜" },
    { label: "葡萄", value: "葡萄" },
  ]);

  return (
    <div className="basic-wrapper">
      <Checkbox
        required
        errMsg="请选择一种食物哦~"
        label="食物"
        options={foods}
      ></Checkbox>
    </div>
  );
};

export default Required;
```

</details>
</Card>

## Checkbox Props

| 属性名 | 类型 | 可选值 | 描述 |
| ---- | ---- | ---- | ---- |
| width | `any` | - | 设置组件的宽度。 |
| valueKey | `string` | - | 设置值的键名。 |
| labelKey | `string` | - | 设置标签的键名。 |
| returnType | `string` | `str / obj` | - | 设置返回值的类型。 |
| suffixContentType | `string` | - | 设置后缀内容的类型。 |
| suffixContent | `any` | - | 设置后缀内容。 |
| name | `string` | - | 设置组件的名称。 |
| isFormItem | `boolean` | - | 控制组件是否为表单项，不是则没有下边距。 |
| errMsg | `string` | - | 设置错误信息。 |
| labelWidth | `any` | - | 设置标签的宽度。 |
| commonSuffixIcon | `string` | - | 设置通用后缀图标。 |
| readOnly | `boolean` | - | 控制组件是否为只读。 |
| label | `string` | - | 设置组件的标签。 |
| labelPosition | `string` | center  / left-top / top | 设置标签的位置，`left-top` 为左上 |
| labelColor | `string` | - | 设置标签的颜色。 |
| required | `boolean` | - | 控制组件是否为必填项。 |
| defaultValue | `string` | string[] | `any[]` | - | 设置默认选中值。 |
| externalClassName | `string` | - | 设置外部类名。 |
| options | `any[]` | - | 设置多选框的选项。 |
| inline | `boolean` | - | 控制选项是否为行内显示。 |
| wrap | `boolean` | - | 控制选项是否换行。 |
| onChange | `(item: any) => void` | - | 选中值变化时的回调函数。 |
| onFormDataChange | `() => void` | - | 表单数据变化时的回调函数。 |
| onFieldChange | `() => void` | - | 字段变化时的回调函数。 |
| onValidateField | `() => void` | - | 字段验证时的回调函数。 |
