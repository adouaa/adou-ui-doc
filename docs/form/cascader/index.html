Cascader 级联选择器

  • 注意：该组件还未使用，等使用后再进行文档完善~

Cascader 是一个级联选择组件，支持多级数据选择，适用于需要从多个层级中选择数据的场景。

2. options​

通过 options 属性，可以设置级联选择的数据源。

2.1 options​

介绍： options 属性用于设定级联选择的数据源，数据源应为一个对象数组，每个对象可以包含 label、value 和 children 属性。

属性​

属性名类型可选值描述
widthany-设置组件的宽度。
inlineboolean-控制组件是否为行内元素。
activeColor{ font: string; bgc: string }-设置选中项的字体和背景颜色。
returnTypestring-设置返回值的类型。
requiredboolean-控制组件是否为必填项。
namestring-设置组件的名称。
labelany-设置组件的标签。
errMsgstring-设置错误信息。
isFormItemboolean-控制组件是否为表单项。
labelColorstring-设置标签的颜色。
showLabelboolean-控制是否显示标签。
labelWidthany-设置标签的宽度。
labelPosition`"center""left-top""top"`
clearableboolean-控制是否可以清除选中项。
valueKeystring-设置值的键名。
labelKeystring-设置标签的键名。
filterableboolean-控制是否可以过滤选项。
optionsany-设置级联选择的数据源。
defaultValuestring[]-设置默认选中值。
onChange(value: string[]) => void-选中值变化时的回调函数。
onFormDataChange(key: string, value: any) => void-表单数据变化时的回调函数。
