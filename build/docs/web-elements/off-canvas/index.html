Drawer 抽屉组件

Drawer 是一个可以从屏幕边缘滑出的抽屉组件，支持 多个方向、自定义触发器、遮罩层、键盘ESC关闭 等特性。常用于展示临时的、补充的内容，如导航菜单、筛选面板等。

1. 基本使用​

最基本的用法，从右侧滑出的抽屉面板。

效果示例：

基础示例

这是一个基础的抽屉示例

点击遮罩层或关闭按钮可以关闭抽屉

点击查看代码
import React from "react";
import Drawer from "@site/src/components/web-elements/adou-off-canvas";
const Basic = () => {
  return (
    <Drawer
      title="基础示例"
      trigger={<button className="btn btn-primary">打开抽屉</button>}
    >
      <div>
        <p>这是一个基础的抽屉示例</p>
        <p>点击遮罩层或关闭按钮可以关闭抽屉</p>
      </div>
    </Drawer>
  );
};
export default Basic;

2. 不同方向​

通过 placement 属性可以控制抽屉从不同方向滑出。

效果示例：

左侧抽屉

从左侧滑出的抽屉

顶部抽屉

从顶部滑出的抽屉

点击查看代码
import React from "react";
import Drawer from "@site/src/components/web-elements/adou-off-canvas";
const Placement = () => {
  return (
    <div className="d-flex gap-2">
      <Drawer
        placement="start"
        title="左侧抽屉"
        trigger={<button className="btn btn-primary">从左侧打开</button>}
      >
        <p>从左侧滑出的抽屉</p>
      </Drawer>
      <Drawer
        placement="top"
        height="300px"
        title="顶部抽屉"
        trigger={<button className="btn btn-primary">从顶部打开</button>}
      >
        <p>从顶部滑出的抽屉</p>
      </Drawer>
    </div>
  );
};
export default Placement;

3. 自定义触发器​

通过 trigger 属性可以自定义触发器，通过 ref 可以在外部控制抽屉的开关。

效果示例：

自定义触发器

使用自定义的触发器打开抽屉

点击查看代码
import React, { useRef } from "react";
import Drawer from "@site/src/components/web-elements/adou-off-canvas";
const CustomTrigger = () => {
  const offCanvasRef = useRef<any>();
  return (
    <div>
      <Drawer
        actRef={offCanvasRef}
        title="自定义触发器"
        trigger={
          <div className="d-flex gap-2">
            <button className="btn btn-primary">触发器1</button>
            <button className="btn btn-secondary">触发器2</button>
          </div>
        }
      >
        <p>使用自定义的触发器打开抽屉</p>
      </Drawer>
      <button
        className="btn btn-link"
        onClick={() => offCanvasRef.current.open()}
      >
        外部打开抽屉
      </button>
    </div>
  );
};
export default CustomTrigger;

4. 禁用点击外部关闭​

通过设置 clickOutside={false} 以及 maskClosable={false} 可以禁用点击外部区域关闭抽屉的功能。

效果示例：

Offcanvas
禁用点击外部关闭

只能通过关闭按钮关闭抽屉

点击查看代码
import Drawer from "@site/src/components/web-elements/adou-off-canvas";
import React from "react";
const NoClickOutside = () => {
  return (
    <Drawer
      clickOutside={false}
      maskClosable={false}
      trigger={<button className="btn btn-primary">打开抽屉</button>}
    >
      <div className="p-3">
        <h5>禁用点击外部关闭</h5>
        <p>只能通过关闭按钮关闭抽屉</p>
      </div>
    </Drawer>
  );
};
export default NoClickOutside;

抽屉内可以放置任意内容，如表单、列表等。

import Drawer from "@site/src/components/web-elements/adou-off-canvas";
import React from "react";
const CustomContent = () => {
  return (
    <Drawer trigger={<button className="btn btn-primary">打开抽屉</button>}>
      <div className="p-3">
        <h5>自定义内容</h5>
        <form>
          <div className="mb-3">
            <label className="form-label">邮箱</label>
            <input type="email" className="form-control" />
          </div>
          <div className="mb-3">
            <label className="form-label">密码</label>
            <input type="password" className="form-control" />
          </div>
          <button type="submit" className="btn btn-primary">
            提交
          </button>
        </form>
      </div>
    </Drawer>
  );
};
export default CustomContent;

属性​

组件属性​

属性名类型默认值描述
titleReact.ReactNode'Offcanvas'抽屉标题。
placement'start' | 'end' | 'top' | 'bottom''end'抽屉滑出的方向。
widthstring | number'400px'抽屉宽度（左右方向时有效）。
heightstring | number'100%'抽屉高度（上下方向时为宽度）。
showMaskbooleantrue是否显示遮罩层。
maskClosablebooleantrue点击遮罩层是否可以关闭。
showCloseButtonbooleantrue是否显示关闭按钮。
clickOutsidebooleantrue是否可以点击外部区域关闭。
classNamestring''抽屉容器的自定义类名。
headerClassNamestring''抽屉头部的自定义类名。
bodyClassNamestring''抽屉内容区的自定义类名。
triggerReact.ReactNode-自定义触发器。
onClose() => void-抽屉关闭时的回调。
onOpen() => void-抽屉打开时的回调。

ref 方法​

组件通过 actRef 或 ref 暴露以下方法：

方法名说明参数返回值
open打开抽屉。-void
close关闭抽屉。-void
toggle切换抽屉开关状态。-void
isOpen获取当前抽屉是否打开。-boolean

注意事项​

  1. 动画时序：

    • 打开时：先显示遮罩层（100ms）后显示抽屉
    • 关闭时：先隐藏抽屉（300ms）后隐藏遮罩层（500ms）

  2. 样式覆盖：

    • 可以通过 className、headerClassName、bodyClassName 自定义样式
    • 宽度和高度可以通过 width 和 height 属性控制

  3. 可访问性：

    • 组件包含适当的 ARIA 属性
    • 支持键盘操作

  4. 性能优化：

    • 使用 useClickOutside hook 处理外部点击
    • 动画使用 CSS transform 实现
