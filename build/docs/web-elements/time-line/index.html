TimeLine 时间轴

TimeLine 组件用于展示时间流或步骤流程。

1. 基本使用​

最基础的垂直时间轴。

2024-01-01
事件1
这是事件1的描述
2024-01-02
事件2
2024-01-03
事件3
点击查看代码
import React from "react";
import TimeLine, {
  TimeLineItem,
} from "@site/src/components/web-elements/adou-time-line";

const events = [
  {
    id: 1,
    title: "事件1",
    date: "2024-01-01",
    isFinish: true,
    description: "这是事件1的描述",
  },
  { id: 2, title: "事件2", date: "2024-01-02" },
  { id: 3, title: "事件3", date: "2024-01-03" },
];

const Basic = () => {
  return (
    <TimeLine events={events}>
      {events.map((event) => (
        <TimeLineItem key={event.id} />
      ))}
    </TimeLine>
  );
};

export default Basic;

2. 水平时间轴​

通过设置 vertical={false} 可以展示水平方向的时间轴。

2022-01-01 12:00:00
步骤1
2022-01-01 12:00:00
步骤2
2022-01-01 12:00:00
步骤3
点击查看代码
import TimeLine, {
  TimeLineItem,
} from "@site/src/components/web-elements/adou-time-line";
import React from "react";

const events = [
  { id: 1, title: "步骤1", isFinish: true, date: "2022-01-01 12:00:00" },
  { id: 2, title: "步骤2", isFinish: true, date: "2022-01-01 12:00:00" },
  { id: 3, title: "步骤3", date: "2022-01-01 12:00:00" },
];
const Horizontal = () => {
  return (
    <TimeLine events={events} vertical={false}>
      {events.map((event) => (
        <TimeLineItem key={event.id} />
      ))}
    </TimeLine>
  );
};

export default Horizontal;

3. 自定义图标​

通过 checkedIcon 和 unCheckedIcon 分别可以控制 完成 和 未完成 状态下的图标。

2023-01-01
Event 1
Description for event 1
2023-02-01
Event 2
Description for event 2
2023-03-01
Event 3
Description for event 3
点击查看代码
import React from "react";
import TimeLine, {
  TimeLineItem,
} from "@site/src/components/web-elements/adou-time-line";

const events = [
  {
    id: 1,
    title: "Event 1",
    description: "Description for event 1",
    date: "2023-01-01",
    isFinish: true,
  },
  {
    id: 2,
    title: "Event 2",
    description: "Description for event 2",
    date: "2023-02-01",
    unCheckedIcon: "fa-solid fa-xmark text-danger",
  },
  {
    id: 3,
    title: "Event 3",
    description: "Description for event 3",
    date: "2023-03-01",
  },
];

const CustomIcon = () => {
  return (
    <TimeLine events={events}>
      {events.map((event) => (
        <TimeLineItem key={event.id} />
      ))}
    </TimeLine>
  );
};
export default CustomIcon;

4. 自定义渲染​

通过 render 函数可以自定义每个时间节点的渲染内容。

2024-01-11 23:47
bug修复
修复了若干bug
2024-01-11 15:47
发布版本
发布 v1.0.0 版本
点击查看代码
import TimeLine, {
  TimeLineItem,
} from "@site/src/components/web-elements/adou-time-line";
import React from "react";

const events = [
  {
    date: "2024-01-11 23:47",
    id: 1,
    title: "bug修复",
    content: "修复了若干bug",
    isFinish: true,
  },
  {
    date: "2024-01-11 15:47",
    id: 1,
    title: "发布版本",
    content: "发布 v1.0.0 版本",
    isFinish: true,
    render: (data) => (
      <>
        <div className="custom-timeline-item pb-1">
          <div className="title d-flex align-items-center">
            <div className="text text-danger me-2">{data.title}</div>
            <div className={`fa-regular fa-thumbs-up text-danger`}></div>
          </div>
          <div className="content-text">{data.content}</div>
        </div>
      </>
    ),
  },
  // ... 其他事件
];
const CustomRender = () => {
  return (
    <TimeLine events={events}>
      {events.map((event) => (
        <TimeLineItem
          key={event.id}
          render={(data) => (
            <div className="custom-timeline-item pb-2">
              <div className={`dot ${data.type}`}></div>
              <div>{data.title}</div>
              <div>{data.content}</div>
            </div>
          )}
        />
      ))}
    </TimeLine>
  );
};

export default CustomRender;

属性​

TimeLine 属性​

属性名类型默认值描述
eventsany[]-时间轴数据源。
verticalbooleantrue是否垂直展示。
childrenReactNode-子组件。

TimeLineItem 属性​

属性名类型默认值描述
dataany-当前节点的数据。
verticalbooleantrue是否垂直展示。
isLastedboolean-是否为最后一个节点。
render(data: any, width: number, height: number, isLasted: boolean) => ReactNode-自定义渲染函数。

render 函数参数​

参数名类型描述
dataany当前节点的数据对象。
