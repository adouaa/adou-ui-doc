useDrag 拖拽钩子

useDrag 是一个用于实现元素拖拽功能的 React Hook。它提供了简单的方式来实现元素的拖拽移动，支持普通元素和对话框两种模式。

1. 基本使用​

最基本的元素拖拽功能。

效果示例：

拖拽我
点击查看代码
import React, { useRef } from "react";
import useDrag from "@site/src/hooks/useDrag";

const Basic = () => {
  const elementRef = useRef(null);
  const triggerRef = useRef(null);
  const { position, handleMouseDown } = useDrag(elementRef, elementRef);
  return (
    <div
      ref={elementRef}
      onMouseDown={handleMouseDown}
      className="draggable-box p-2"
      style={{ width: "fit-content", background: "red" }}
    >
      <div ref={triggerRef} className="drag-handle">
        拖拽我
      </div>
    </div>
  );
};

export default Basic;

2. 自定义样式​

可以取消自动样式设置， 手动控制拖拽元素的样式 。例如手动控制拖拽元素只能 在父组件的内部进行拖拽 。

效果示例：

拖拽元素
点击查看代码
import React, { useRef } from "react";
import useDrag from "@site/src/hooks/useDrag";

const CustomStyle = () => {
  const elementRef = useRef(null);
  const triggerRef = useRef(null);
  const { position, handleMouseDown } = useDrag(
    triggerRef,
    elementRef,
    false,
    false
  );
  return (
    <div
      className="wrapper"
      style={{ backgroundColor: "red", width: "300px", padding: "15px 0" }}
    >
      <div
        className="position-relative"
        ref={elementRef}
        style={{
          top:
            position.y <= -15
              ? "-15px"
              : position.y >= 15
              ? "15px"
              : position.y + "px",
          left:
            position.x <= 0
              ? 0
              : position.x >= 200
              ? "200px"
              : position.x + "px",
          transition: "transform 0.1s",
          cursor: "move",
          zIndex: 999,
        }}
      >
        <div
          className="ps-2"
          ref={triggerRef}
          onMouseDown={handleMouseDown}
          style={{ backgroundColor: "skyblue", width: "100px" }}
        >
          拖拽元素
        </div>
      </div>
    </div>
  );
};

export default CustomStyle;

参数​

参数名类型默认值描述
triggerRefRefObject-触发拖拽的元素引用。
elementRefRefObject-被拖拽的元素引用。
isDialogbooleanfalse是否为对话框模式。
autoStylebooleantrue是否自动设置样式。
initialPosition{ x: number, y: number }{ x: 0, y: 0 }初始位置。

返回值​

属性名类型描述
position{ x: number, y: number }当前位置坐标。
handleMouseDown(e: MouseEvent) => void鼠标按下事件处理函数。

特性​

  1. 两种模式：

    • 支持普通元素拖拽
    • 支持对话框拖拽

  2. 样式控制：

    • 可选择自动样式设置
    • 支持手动样式控制

  3. 位置管理：

    • 自动计算相对位置
    • 支持设置初始位置

注意事项​

  1. 引用设置：

    • 必须为触发元素和目标元素设置 ref
    • triggerRef 和 elementRef 可以是同一个元素

  2. 样式建议：

    • 使用 autoStyle 时，确保元素可以使用相对定位
    • 关闭 autoStyle 时，需要手动处理位置样式

  3. 性能考虑：

    • 拖拽过程中会持续触发状态更新
    • 建议使用防抖或节流优化频繁更新
